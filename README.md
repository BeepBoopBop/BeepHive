BeepHive
========

Swarm Robotics API and Simulator

## Build Instructions
### Make and enter build directory:
```
$ mkdir build
$ cd build
```
### Configure the build system
```
$ cmake ..
```
### Build the project
```
$ make
```
### Build documetation
```
$ make doc
```
## Dependencies
### Overture
