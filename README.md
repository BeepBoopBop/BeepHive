BeepHive
========

Swarm Robotics API and Simulator

## Build Instructions
### Make and enter build directory:
```
$ mkdir build
$ cd build
```
### Configure the build system
```
$ cmake -D CMAKE_CXX_COMPILER=mpicxx -D CMAKE_C_COMPILER=mpicc ..
```
### Build the project
```
$ make
```
### Build documetation
```
$ make doc
```
